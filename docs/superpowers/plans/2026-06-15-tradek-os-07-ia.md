# TradeK OS — Plano 07: Agente IA real + Relatórios (executado)

**Goal:** Substituir o fluxo scriptado do agente por **Claude** real (conversa, classifica, dá score, cria lead) e gerar relatórios IA. Stack: `@anthropic-ai/sdk` em Edge Functions Deno.

## Entregue
- **Edge Function `agent-chat`** (`--no-verify-jwt`): Claude (`claude-opus-4-8`, env `AGENT_MODEL`) com tool use — `buscar_produtos` (catálogo real do banco, respeitando `permitir_cotacao_ia`) e `registrar_lead` (cria company+contact+lead+report+interaction, com score e classificação). System prompt com guardrails do spec 08.
- **Edge Function `generate-report`** (JWT): relatório IA do lead com **adaptive thinking** (`thinking:{type:"adaptive"}`, `effort:"high"`), versionado em `reports`.
- **Frontend:** `AgentWidget` virou **chat livre** com a IA (`callAgent` → `agent-chat`), com fallback gentil se a chave faltar. Botão **"Gerar"** na aba Relatório do `LeadModal` → `generate-report`.
- **Segredo:** `ANTHROPIC_API_KEY` setado nos Function secrets (fora do front/git).

## Verificação E2E (ao vivo)
- `agent-chat`: conversa em PT-BR → classificou SCF → criou lead com **score 85 / "quente" / qualificado**, `o_que_quer` e `resumo_ia` preenchidos pelo Claude; relatório inline gerado. Pelo navegador: tool `buscar_produtos` devolveu as 7 motos reais e aplicou o guardrail de preço.
- `generate-report`: relatório estruturado completo (Resumo executivo, seções do spec 08) com adaptive thinking.
